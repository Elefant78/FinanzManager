# Screenshots

Hier kommen Screenshots der laufenden Anwendung rein. Erwartete Dateinamen:

- `dashboard.png` — die Monatsübersicht mit Statistik-Kacheln
- `buchungen.png` — die Buchungsliste mit Filter
- `kategorien.png` — die Kategorien-Verwaltung

Empfohlen: PNG-Format, Fensterbreite ca. 1100 px (Standardgrösse der App).

So erstellt man unter Windows einen sauberen Screenshot:

1. App starten und die gewünschte Ansicht öffnen.
2. `Alt + Druck` (Alt + PrintScreen) — kopiert nur das aktive Fenster.
3. In Paint einfügen (`Strg + V`) und als PNG speichern.

Alternativ das Snipping Tool (`Windows + Shift + S`) verwenden.
